자주 묻는 질문 (FAQ)

한국사 10분 챌린지는 무료인가요?

네. 한국사 10분 챌린지는 회원가입 없이 무료로 이용할 수 있는 한국사 퀴즈 서비스입니다. 메인 퀴즈, 오답노트, 학습 분석, 핵심 연표, 비교 정리 페이지를 자유롭게 볼 수 있습니다. 짧게 실력을 점검하고 싶은 사용자도 부담 없이 시작할 수 있도록 구성했습니다.

한국사능력검정시험(한능검) 대비에 도움이 되나요?

한능검에서 자주 다루는 시대 흐름, 제도, 인물, 사건을 중심으로 문제와 해설을 구성하고 있습니다. 기출 문항을 그대로 옮기는 방식이 아니라, 중요한 개념을 직접 만든 문항으로 다시 확인할 수 있게 하는 방향입니다. 시험 전에는 10문제 퀴즈로 약한 시대를 확인하고, 오답노트와 비교 정리로 복습하는 방식으로 활용하면 좋습니다.

수능 한국사 대비가 가능한가요?

수능 한국사에서 중요한 전체 흐름과 핵심 개념을 점검하는 데 활용할 수 있습니다. 수능 한국사는 세부 암기보다 시대별 흐름과 사건의 의미를 잡는 것이 중요하므로, 퀴즈를 풀고 틀린 문제의 해설을 확인한 뒤 핵심 연표로 순서를 다시 보는 흐름을 추천합니다.

오답노트는 어떻게 사용하나요?

퀴즈를 풀다가 틀린 문제는 자동으로 오답노트에 저장됩니다. 오답노트에서는 틀렸던 문제를 다시 확인하고, 필요하면 랜덤 복습으로 다시 풀어볼 수 있습니다. 단순히 점수만 보고 끝내기보다 틀린 이유를 다시 보는 용도로 사용하면 복습 효과가 더 좋습니다.

학습 분석은 무엇인가요?

학습 분석은 지금까지 푼 문제를 바탕으로 시대별 정답률을 보여주는 기능입니다. 어느 시대는 안정적으로 맞히고, 어느 시대에서 자주 틀리는지 확인할 수 있어 복습 우선순위를 정하는 데 도움이 됩니다. 분석이 너무 적은 데이터로 흔들리지 않도록 일정량 이상 문제를 푼 뒤 확인하는 방식으로 운영하고 있습니다.

학습 기록은 어디에 저장되나요?

학습 기록은 현재 사용 중인 브라우저에 저장됩니다. 브라우저 데이터를 삭제하면 기록이 초기화될 수 있습니다. 별도의 회원 정보를 수집해 서버에 저장하는 방식이 아니기 때문에, 다른 기기나 다른 브라우저에서는 기존 기록이 이어지지 않을 수 있습니다.

회원가입 없이 이용할 수 있나요?

네. 별도의 회원가입이나 로그인 없이 바로 문제를 풀 수 있습니다. 처음 방문한 사용자도 메인 화면에서 바로 10문제 퀴즈를 시작할 수 있습니다. 학습 기록은 사용 중인 브라우저에 저장되므로, 같은 브라우저에서 이어서 복습하는 방식이 가장 안정적입니다.

모바일에서도 이용할 수 있나요?

네. 스마트폰, 태블릿, PC에서 모두 이용할 수 있도록 반응형 화면으로 구성했습니다. 특히 검색으로 비교 정리 페이지에 들어오는 사용자가 많아, 모바일에서도 표와 설명을 최대한 읽기 쉽게 정리하고 있습니다. 다만 기기 폭이 매우 좁은 경우 일부 표는 가로 스크롤로 보는 방식이 더 편할 수 있습니다.

틀린 문제만 다시 풀 수 있나요?

가능합니다. 틀린 문제는 오답노트에 자동 저장되고, 오답 문제만 따로 다시 확인할 수 있습니다. 랜덤 복습 기능을 사용하면 이전에 틀렸던 문제를 다시 풀어보며 기억이 남아 있는지 점검할 수 있습니다. 오답이 쌓일수록 자신이 반복해서 헷갈리는 시대나 개념을 파악하기 쉬워집니다.

문제는 계속 추가되나요?

네. 문제와 해설은 사이트 운영 과정에서 계속 검토하고 보완하고 있습니다. 다만 단순히 문제 수만 늘리기보다, 너무 쉬운 보기 구성이나 애매한 해설이 없도록 품질을 함께 확인하려고 합니다. 새로운 비교 정리 주제와 연표 보강도 이용자가 실제로 헷갈릴 만한 내용 위주로 추가하고 있습니다.

문제는 직접 만든 문항인가요?

네. 한국사 10분 챌린지의 문제와 해설은 학습용으로 직접 작성하고 있습니다. 한능검과 수능 한국사에서 중요하게 다루는 시대와 개념을 참고하되, 특정 시험의 기출 문항이나 보기 구성을 그대로 옮기지 않도록 주의하고 있습니다. 문항을 만들 때는 정답이 너무 쉽게 드러나지 않도록 같은 시대나 비슷한 개념의 선택지를 우선 고려합니다.

연표와 비교 정리는 어떻게 활용하면 좋나요?

연표는 사건의 순서와 흐름을 잡을 때 활용하면 좋습니다. 예를 들어 개항 이후 개화 정책, 임오군란, 갑신정변, 동학 농민 운동, 갑오개혁처럼 앞뒤 관계가 중요한 사건은 연표로 먼저 흐름을 확인하는 것이 도움이 됩니다.

비교 정리는 이름이나 성격이 비슷해서 헷갈리는 개념을 따로 정리할 때 유용합니다. 거란·여진·몽골, 훈구와 사림, 대동법·균역법·영정법처럼 자주 섞이는 주제는 비교표와 핵심 해설을 함께 보면 기억하기 쉽습니다.

문제 오류나 오탈자는 어디에 제보하나요?

문제 오류, 오탈자, 해설 보완 의견은 문의 페이지의 이메일로 보내주시면 됩니다. 가능하면 문제에 나온 시대, 질문 문장, 선택지, 정답으로 표시된 내용, 오류라고 생각한 이유를 함께 적어 주세요. 보내주신 내용은 확인 후 문제 수정과 해설 보완에 참고합니다.

어떤 순서로 공부하면 좋나요?

처음 방문했다면 먼저 메인 화면에서 전체 문제 10문제를 풀어 현재 실력을 가볍게 점검해보세요. 그다음 결과 화면에서 오답노트와 학습 분석을 확인하면 약한 시대와 자주 틀리는 유형을 파악할 수 있습니다.

약한 시대가 보이면 핵심 연표로 사건의 순서를 다시 보고, 비슷한 제도나 사건이 헷갈린다면 비교 정리 페이지에서 차이를 확인하는 흐름을 추천합니다. 짧은 퀴즈, 오답 확인, 연표 복습, 비교 정리를 반복하면 단순 암기보다 안정적으로 한국사 흐름을 익힐 수 있습니다.
